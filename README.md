#  All-in-One Scientific Calculator

## 📌 Overview

The **All-in-One Scientific Calculator** is a beginner-friendly Python project designed to perform a wide range of calculations within a single application. Instead of creating separate programs for different tasks, this calculator integrates **basic arithmetic, scientific, and health-related calculations** into one easy-to-use system.

This project serves as an excellent introduction to Python programming and demonstrates how real-world applications can be built using fundamental programming concepts.

---

##  Features

###  1. Basic Arithmetic Operations

* Addition, subtraction, multiplication, division
* Power calculations
* Square root calculations
* Error handling (e.g., division by zero)

###  2. Trigonometric Functions

* Sine (sin), Cosine (cos), Tangent (tan)
* Accepts angles in **degrees**
* Handles special angles like 0°, 90°, 180°, 270°

### 📊 3. Logarithmic Functions

* Natural logarithm (ln)
* Log base 10
* Input validation for invalid values

###  4. Health Calculations

* Body Mass Index (BMI) calculator
* Body Fat Percentage calculator
* Displays **health status categories** (e.g., underweight, normal, overweight)

---

##  Methodology

This project uses a **modular programming approach**, meaning:

* Each operation is implemented as a **separate function**
* Improves readability, testing, and future improvements

### Key Programming Concepts Used:

*  Functions (for structured code)
*  Loops (to keep the program running)
*  Conditional statements (for decision making)
* Dictionary-based menu system (for easy expansion)

Instead of relying on advanced libraries, mathematical logic like:

* Taylor Series (for sine & cosine)
* Series expansion (for logarithms)

was implemented manually to strengthen understanding.

---

##  How It Works

The calculator follows a **menu-driven system**:

1. Program displays a list of operations
2. User selects an option
3. Program asks for input values
4. Selected function runs
5. Result is displayed

###  Basic Flow (Pseudocode)

```
START
WHILE True:
    Display menu
    Take user input

    IF user chooses exit:
        End program

    IF invalid option:
        Show error

    ELSE:
        Take required inputs
        Call function
        Display result
END
```
